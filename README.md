Stock Price Predictor 📈

A modern AI-powered stock market prediction web application built with React, TypeScript, Vite, and Gemini AI. The platform provides stock price forecasting, technical analysis, backtesting, portfolio simulation, interactive charts, market news insights, and AI-generated predictions for popular stocks and cryptocurrencies.

Features
📊 Interactive stock price charts
🤖 AI-powered forecasting using Gemini AI
📈 Multiple prediction models (Linear Regression, Moving Averages, etc.)
🔍 Technical analysis indicators
💼 Virtual portfolio management
🧪 Backtesting of trading strategies
📰 Market news and insights
₿ Cryptocurrency support (BTC-USD)
⚡ Fast and responsive React UI
Tech Stack
React 19
TypeScript
Vite
Tailwind CSS
Gemini AI API
Express.js
